Project 1
GITHUB LINK

Bus Network Management System

Description

A C++ console-based application that manages a city’s bus network efficiently. It allows administrators to add stops and routes, while passengers can search for stops, view routes, find the shortest path, and book or cancel tickets. The system solves the problem of manually managing and navigating bus routes by automating data handling and route optimization using Dijkstra’s algorithm.

Technologies used

  • C++ (Core Language)
  • File Handling (CSV data storage)
  • Graph Data Structure (Adjacency List Representation)
  • Dijkstra’s Algorithm (Shortest Path Calculation)
  • STL (Standard Template Library – vectors, maps, priority queue)
  • Console I/O for User Interaction

Bank Management System

Description

A C++ console-based application designed to manage basic banking operations efficiently. It allows users to create accounts, deposit and withdraw money, check balances, and update or delete account information. The system solves the problem of manual record-keeping by storing all customer and transaction data using file handling, ensuring data persistence between program runs.

Technologies used

  • File Handling (CSV data storage)
  • Object-Oriented Programming (Classes and Objects)
  • Functions and Data Structures (for Account Management)
  • STL (Standard Template Library – vectors, maps, priority queue)
  • Console I/O for User Interaction
